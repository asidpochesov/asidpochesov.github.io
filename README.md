1. **HTML** нужен для структуры страницы, CSS — для её внешнего вида. HTML задаёт что показывать, CSS — как это выглядит.
2. **GitHub Pages** — бесплатный хостинг от GitHub. Берёт файлы из репозитория и публикует их как сайт по адресу логин.github.io.
3. Зависит от того что использовал в работе, но стандартный ответ:

**HTML теги:**
- `<h1>` — заголовок первого уровня
- `<p>` — абзац текста
- `<a>` — ссылка
- `<ul>` / `<li>` — маркированный список
- `<nav>` — навигационное меню
- `<header>` / `<footer>` — шапка и подвал

**CSS свойства:**
- `color` — цвет текста
- `background-color` — цвет фона
- `font-family` — шрифт
- `margin` / `padding` — отступы
- `display: flex` — гибкое расположение элементов
